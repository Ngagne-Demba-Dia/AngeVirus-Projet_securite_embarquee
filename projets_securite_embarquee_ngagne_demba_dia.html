ASCAD · Attaques sur traces reelles avec SCALib et scared

ASCAD (ANSSI Side-Channel Analysis Database) est le dataset de reference pour apprendre les attaques side-channel. Il contient 60 000 traces de puissance d'un AES-128 masque sur STM32, publiees par l'ANSSI avec les cles, textes clairs et masques. C'est le terrain d'entrainement standard des chercheurs en securite materielle.

Reference : Benadjila et al., "ASCAD: ANSSI SCA Database", 2018 — github.com/ANSSI-FR/ASCAD
Raspberry Pi comme serveur SCALib : SCALib compile nativement sur ARM64 (Pi 4 en mode 64-bit). Toutes les cellules de code de cette section tournent sans modification sur le Pi. Le jeu de donnees ASCAD (~2.3 Go) se telecharge directement sur le Pi via wget. Le Pi 4 (4 Go RAM) traite 50 000 traces SNR en ~10 s.

3. CPA avec scared — attaque non profilee (et son echec sur ASCAD masque)

scared (Eshard) est la librairie Python SCA la plus orientee praticien. Elle fournit CPAAttack, DPAAttack, des modeles de fuite (HammingWeight, Value), et un systeme de conteneur de traces (estraces). L'objectif ici est de montrer pourquoi la CPA echoue sur ASCAD masque, et comment scared permet de l'exprimer proprement.

import scared
import estraces   # pip install estraces  (dependance de scared)
import numpy as np
import matplotlib.pyplot as plt

# ── 1. Creer le TraceHeaderSet (conteneur scared) ──────────────────────────
# estraces.read_ths_from_ram accepte les traces + les metadonnees comme kwargs
ths_att = estraces.read_ths_from_ram(
    traces    = X_att.astype(np.float32),   # (10000, 700)
    plaintext = pt_att                       # (10000, 16) uint8
)

# ── 2. Definir la fonction de selection (modele intermediaire) ─────────────
@scared.attack_function
def sbox_hw(plaintext, guesses):
    """
    plaintext : (batch, 16) uint8   — batch de textes clairs
    guesses   : (256,) uint8        — hypotheses de cle byte
    Retourne  : (batch, 256) uint8  — HW(SBox[pt[TARGET] XOR guess])
    """
    pt  = plaintext[:, TARGET].astype(np.uint16)          # (batch,)
    g   = guesses.astype(np.uint16)                        # (256,)
    inter = SBOX[(pt[:, None] ^ g[None, :]) & 0xFF]        # (batch, 256)
    hw_lut = np.array([bin(v).count('1') for v in range(256)], dtype=np.uint8)
    return hw_lut[inter]                                    # (batch, 256) uint8

# ── 3. Lancer la CPA ───────────────────────────────────────────────────────
attack = scared.CPAAttack(
    selection_function = sbox_hw,
    model              = scared.HammingWeight(),
    discriminant       = scared.nanmax,        # garder la corr max sur le temps
    convergence_step   = 200,                  # calculer le rang tous les 200 traces
)
attack.run(ths_att)

# ── 4. Resultats ───────────────────────────────────────────────────────────
# attack.results   : (256, 700) — carte de correlation par hypothese et par echantillon
# attack.scores    : (256,)     — correlation max par hypothese
# attack.convergence_traces : courbe de rang de chaque hypothese au cours du temps

best_guess = int(np.argmax(attack.scores))
rank_true  = int(np.where(np.argsort(-attack.scores) == TRUE_KEY[TARGET])[0][0]) + 1

print(f"Meilleure hypothese scared CPA : 0x{best_guess:02X}")
print(f"Vraie cle octet {TARGET}        : 0x{TRUE_KEY[TARGET]:02X}")
print(f"Rang de la vraie cle           : {rank_true}/256")
print("→ rang eleve attendu : le masking brise la correlation CPA directe")

# Visualiser la carte de correlation (toutes hypotheses, tous echantillons)
plt.figure(figsize=(14, 5))
plt.imshow(np.abs(attack.results), aspect='auto', cmap='hot',
           vmin=0, vmax=np.percentile(np.abs(attack.results), 99))
plt.colorbar(label='|Correlation|')
plt.xlabel('Echantillon temporel')
plt.ylabel('Hypothese de cle (0-255)')
plt.axhline(TRUE_KEY[TARGET], color='cyan', linewidth=1, label=f'Vraie cle 0x{TRUE_KEY[TARGET]:02X}')
plt.legend(); plt.title('CPA ASCAD masque — scared — correlation map')
plt.savefig('ascad_cpa_scared.png', dpi=150)
Pourquoi la CPA echoue sur ASCAD : ASCAD implemente un AES masque au premier ordre. La puce calcule SBox[pt XOR k] XOR mask ou mask est uniforme et different a chaque trace. Comme HW(v XOR mask) est statistiquement independant de HW(v) quand mask est uniforme, la correlation entre HW(SBox[pt XOR k]) et la trace disparait. Cela est visible dans la carte de correlation : aucune hypothese ne ressort clairement.

4. Attaque profilee : LDA + Template avec SCALib

SCALib implemente une attaque template efficace en deux phases : (1) LDA sur les traces de profiling (masques connus) pour apprendre un modele gaussien de la fuite ; (2) scoring des traces d'attaque (masques inconnus) par accumulation des log-probabilites. C'est l'approche classique qui fonctionne sur ASCAD masque.

from scalib.modeling import LDAClassifier
import numpy as np
import matplotlib.pyplot as plt
from tqdm import tqdm

N_TRAIN = 45000   # 45k pour entrainement, 5k pour validation

# ── 1. Labels : valeur masquee sur le set de profiling ────────────────────
# L'implementation ASCAD calcule SBox[pt XOR k] XOR masks[0] pour le byte TARGET
# On connait masks[0] dans le set de profiling → cible apprise correctement
labels_prof = np.array([
    int(SBOX[int(pt_prof[i, TARGET]) ^ int(key_prof[i, TARGET])]) ^ int(mask_prof[i, 0])
    for i in range(len(X_prof))
], dtype=np.uint16)

X_train = X_prof[:N_TRAIN].astype(np.int16)
y_train = labels_prof[:N_TRAIN].reshape(-1, 1)   # shape (N, 1) uint16

X_val   = X_prof[N_TRAIN:].astype(np.int16)
y_val   = labels_prof[N_TRAIN:]

# ── 2. Entrainement LDA (SCALib) ──────────────────────────────────────────
# LDAClassifier : nc=256 classes, ns=700 echantillons, p=1 variable cible
print("Entrainement LDA + modele gaussien sur 45 000 traces profiling...")
lda = LDAClassifier(nc=256, ns=700, p=1)
lda.fit_u(X_train, y_train)   # traces int16, labels uint16 shape (N,1)
print("Entrainement termine.")

# ── 3. Validation ─────────────────────────────────────────────────────────
# predict_proba retourne les log-probabilites : (N_val, 256)
log_p_val     = lda.predict_proba(X_val)        # (5000, 256)
predicted_val = np.argmax(log_p_val, axis=1)
accuracy      = (predicted_val == y_val).mean()
print(f"Accuracy validation (classe masquee) : {accuracy:.3f}  [> 0.05 = mieux qu'aleatoire]")

# ── 4. Attaque sur les traces d'attaque (masques inconnus) ────────────────
# Strategie : pour chaque hypothese k, scorer avec P(trace | SBox[pt XOR k])
# Le modele LDA appris sur valeurs masquees fait office de "detecteur" de
# l'intermediaire de base, car il a vu toutes les valeurs masquees possibles
# pendant le profiling et apprend la structure sous-jacente.

print("Attaque sur 10 000 traces (masques inconnus)...")
N_ATT = len(X_att)

# Calculer les log-probs en batch (plus rapide que trace par trace)
log_p_att = lda.predict_proba(X_att.astype(np.int16))   # (10000, 256)

# Pour chaque trace i et chaque hypothese k :
# score[i, k] = log_p_att[i, SBox[pt_i[TARGET] XOR k]]
pt_col     = pt_att[:, TARGET].astype(np.uint16)         # (10000,)
guesses    = np.arange(256, dtype=np.uint16)             # (256,)
sbox_inter = SBOX[(pt_col[:, None] ^ guesses[None, :]) & 0xFF]  # (10000, 256)

# Indexer les log-probs : pour chaque trace i, prendre la log-prob
# de la classe SBox[pt_i XOR k] pour chaque hypothese k
log_key_scores = np.take_along_axis(log_p_att, sbox_inter.astype(np.int64), axis=1)
# log_key_scores shape : (10000, 256)

# Cumul : rang de la vraie cle apres n traces
cumulative = np.cumsum(log_key_scores, axis=0)   # (10000, 256)
step = max(1, N_ATT // 200)
ranks_n, ranks_v = [], []
for n in range(step, N_ATT + 1, step):
    order = np.argsort(-cumulative[n - 1])
    ranks_n.append(n)
    ranks_v.append(int(np.where(order == TRUE_KEY[TARGET])[0][0]) + 1)

# ── 5. Courbe de rang ─────────────────────────────────────────────────────
plt.figure(figsize=(10, 4))
plt.plot(ranks_n, ranks_v, color='darkgreen', linewidth=1.5, label='LDA Template')
plt.axhline(1, color='red', linestyle='--', label='Rang 1 = cle retrouvee')
plt.xlabel("Traces d'attaque utilisees"); plt.ylabel("Rang de la vraie cle")
plt.title(f"Template LDA SCALib — octet {TARGET} — ASCAD masque")
plt.yscale('log'); plt.legend(); plt.savefig("ascad_template_rank.png", dpi=150)
print(f"Rang final ({N_ATT} traces) : {ranks_v[-1]}/256")
Resultat attendu : la vraie cle atteint le rang 1 avec 200 a 2000 traces d'attaque selon l'octet cible et la qualite du modele. La courbe de rang descend en escalier vers 1. Si elle stagne, augmenter N_TRAIN ou verifier que les labels utilisent bien le masque correct (masks[i, 0] et non masks[i, TARGET] — verifier la structure exacte du dataset ASCAD).

5. TVLA avec SCALib et scipy

Le TVLA verifie si une implementation fuit, sans avoir besoin de connaitre la cle. SCALib fournit une implementation rapide du t-test de Welch ; scipy est l'alternative si SCALib n'est pas disponible.

from scalib.metrics import Ttest
from scipy import stats as sp_stats
import numpy as np
import matplotlib.pyplot as plt

Le Deep Learning contourne la selection manuelle de points d'interet et resiste naturellement a la desynchronisation. Sur ASCAD, un MLP simple atteint le rang 1 avec moins de 100 traces d'attaque — comparable a l'attaque template LDA, sans avoir besoin de specifier le modele de fuite. 3. Architecture CNN (Zaid et al. 2019)

class CNN_SCA(nn.Module):
    """CNN 1D — plus robuste a la desynchronisation qu'un MLP."""
    def __init__(self, n_samples=700, n_classes=256):
        super().__init__()
        self.conv = nn.Sequential(
            # Bloc 1 : 32 filtres, noyau 11, pool 2 → 700 → 350
            nn.Conv1d(1, 32, kernel_size=11, padding=5), nn.SELU(),
            nn.AvgPool1d(2),
            # Bloc 2 : 64 filtres, noyau 11, pool 2 → 350 → 175
            nn.Conv1d(32, 64, kernel_size=11, padding=5), nn.SELU(),
            nn.AvgPool1d(2),
            # Bloc 3 : 128 filtres, noyau 11, pool 2 → 175 → 87
            nn.Conv1d(64, 128, kernel_size=11, padding=5), nn.SELU(),
            nn.AvgPool1d(2),
        )
        # Taille apres conv : 128 * 87 = 11 136
        self.flat_size = 128 * (n_samples // 8)
        self.head = nn.Sequential(
            nn.Flatten(),
            nn.Linear(self.flat_size, 512), nn.SELU(),
            nn.Linear(512, 256),           nn.SELU(),
            nn.Linear(256, n_classes),
        )

    def forward(self, x):
        x = x.unsqueeze(1)      # (B, 700) → (B, 1, 700) pour Conv1d
        return self.head(self.conv(x))

def train_model(model, loader_train, loader_val, n_epochs=20, lr=1e-3, label='model'):
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    scheduler = torch.optim.lr_scheduler.ReduceLROnPlateau(optimizer, patience=3, factor=0.5)
    criterion = nn.CrossEntropyLoss()

    history = {'train_loss': [], 'val_loss': [], 'val_acc': []}

    for epoch in range(n_epochs):
        t0 = time.time()
        # --- phase train ---
        model.train()
        total_loss = 0.0
        for xb, yb in loader_train:
            xb, yb = xb.to(device), yb.to(device)
            optimizer.zero_grad()
            loss = criterion(model(xb), yb)
            loss.backward()
            optimizer.step()
            total_loss += loss.item() * len(xb)
        train_loss = total_loss / len(loader_train.dataset)

        # --- phase validation ---
        model.eval()
        val_loss, correct = 0.0, 0
        with torch.no_grad():
            for xb, yb in loader_val:
                xb, yb = xb.to(device), yb.to(device)
                logits = model(xb)
                val_loss += criterion(logits, yb).item() * len(xb)
                correct  += (logits.argmax(1) == yb).sum().item()
        val_loss /= len(loader_val.dataset)
        val_acc   = correct / len(loader_val.dataset)

        scheduler.step(val_loss)
        history['train_loss'].append(train_loss)
        history['val_loss'].append(val_loss)
        history['val_acc'].append(val_acc)
        print(f"[{label}] Epoch {epoch+1:3d}/{n_epochs} | "
              f"loss {train_loss:.4f} → {val_loss:.4f} | acc {val_acc:.3f} | "
              f"{time.time()-t0:.1f}s")

    return history

def attack_rank_dl(model, X_att_n, pt_att, TRUE_KEY, TARGET, SBOX,
                   max_traces=2000, step=50):
    """Calcule le rang de la vraie cle pour 1..max_traces traces d'attaque."""
    model.eval()
    X_t = torch.from_numpy(X_att_n[:max_traces]).to(device)

    with torch.no_grad():
        logits = model(X_t)                          # (max_traces, 256)
        log_p  = torch.log_softmax(logits, dim=1).cpu().numpy()

    # Pour chaque hypothese k_guess, mapper les 256 sorties DL vers les classes cibles
    guesses   = np.arange(256, dtype=np.uint16)
    pt_col    = pt_att[:max_traces, TARGET].astype(np.uint16)

    # sbox_inter[i, k] = SBox[pt[i] XOR k]  → classe DL correspondante
    sbox_inter = SBOX[(pt_col[:, None] ^ guesses[None, :]) & 0xFF]   # (max_traces, 256)

    # log_p_key[i, k] = log P(classe sbox_inter[i,k] | trace i)
    log_p_key = np.take_along_axis(log_p, sbox_inter.astype(np.int64), axis=1)

    ranks = []
    for n in range(step, max_traces + 1, step):
        cumul = log_p_key[:n].sum(axis=0)       # (256,) score cumulatif
        rank  = int(np.sum(cumul > cumul[TRUE_KEY]))
        ranks.append((n, rank))

    return ranks

T1 · Timing attack par oracle temporel Facile

But : demontrer qu'une comparaison de secret avec sortie anticipee fuit dans le temps de reponse, et exploiter cette fuite pour retrouver le secret octet par octet sans aucun equipement supplementaire.

Materiel requis : un MCU (STM32 ou ESP32), un cable USB, un PC. Rien d'autre.

Competences visees : timing attack, oracle, analyse statistique, implementation constant-time.

Etapes

  1. Flasher le firmware vulnerable sur le STM32 : une fonction de comparaison de mot de passe avec retour anticipe, exposee sur UART (envoi de candidat 16 octets, reponse 1 octet OK/NOK).
  2. Depuis Python sur PC, mesurer le RTT (round-trip time) de chaque tentative avec time.perf_counter_ns().
  3. Faire une mediane sur 100-300 essais par candidat pour lisser le bruit serie/OS.
  4. Le bon octet en position courante produit un RTT statistiquement plus long. Fixer, avancer au suivant.
  5. Comparer avec une implementation constant-time (XOR accumule, pas de branchement sur le secret).

/* Fonction volontairement vulnerable - demo seulement */
static const char SECRET[] = "S3cr3tK3y_2024!";

int check_password_vulnerable(const char *guess, int len) {
    if (len != 16) return 0;
    for (int i = 0; i < 16; i++) {
        if (guess[i] != SECRET[i]) return 0;  // retour anticipe : fuite !
    }
    return 1;
}

/* Version constant-time a comparer */
int check_password_safe(const char *guess, int len) {
    if (len != 16) return 0;
    uint8_t diff = 0;
    for (int i = 0; i < 16; i++) {
        diff |= (uint8_t)(guess[i] ^ SECRET[i]);  // XOR accumule, pas de branchement
    }
    return (diff == 0) ? 1 : 0;
}

void uart_loop(void) {
    uint8_t buf[16];
    while (1) {
        uart_read_exact(buf, 16);
        uint8_t result = (uint8_t)check_password_vulnerable((char*)buf, 16);
        HAL_UART_Transmit(&huart2, &result, 1, 100);

}

import serial, time, statistics

PORT = 'COM3'       # adapter selon OS (Linux: /dev/ttyUSB0)
BAUD = 115200
N_MEASURES = 200    # mesures par candidat

port = serial.Serial(PORT, BAUD, timeout=1)
time.sleep(0.1)     # attendre reset MCU apres ouverture port

def measure_rtt(candidate: bytes, n: int = N_MEASURES) -> float:
    """Retourne la mediane du RTT en nanosecondes."""
    times = []
    for _ in range(n):
        port.write(candidate)
        t0 = time.perf_counter_ns()
        port.read(1)                            # attendre le byte de reponse
        times.append(time.perf_counter_ns() - t0)
    return statistics.median(times)

found = bytearray(16)
for pos in range(16):
    best_byte, best_time = 0, 0
    for b in range(32, 127):                    # ASCII imprimable
        candidate = bytes(found[:pos]) + bytes([b]) + bytes(16 - pos - 1)
        t = measure_rtt(candidate)

#include "mbedtls/ecdsa.h"
#include "mbedtls/sha256.h"
#include "mbedtls/ecp.h"

/* Cle publique P-256 gravee dans le bootloader (non modifiable apres RDP1) */
static const uint8_t TRUSTED_PUBKEY_X[32] = { 0x... };  // coordonnee X de la cle publique
static const uint8_t TRUSTED_PUBKEY_Y[32] = { 0x... };  // coordonnee Y de la cle publique

#define APP_START    0x08010000U
#define APP_MAX_SIZE (256*1024)
#define SIG_ADDR     0x0803F000U

typedef struct {
    uint32_t magic;        // 0x5E00B001
    uint32_t version;      // numero de version (anti-rollback)
    uint32_t size;         // taille de l'image en octets
    uint8_t  sig_r[32];    // signature ECDSA : composante r
    uint8_t  sig_s[32];    // signature ECDSA : composante s
} __attribute__((packed)) ImageMetadata;

static void halt_secure(void) {
    /* Effacer registres sensibles, boucler indefiniment */
    __DSB(); __ISB();
    while (1) { __WFI(); }
}

void verify_and_boot(void) {
    ImageMetadata *meta = (ImageMetadata*)SIG_ADDR;

    if (meta->magic != 0x5E00B001)       halt_secure();
    if (meta->size  > APP_MAX_SIZE)       halt_secure();

    /* 1. Hacher l'image */
    uint8_t hash[32];
    mbedtls_sha256((uint8_t*)APP_START, meta->size, hash, 0);

    /* 2. Charger la cle publique */
    mbedtls_ecdsa_context ctx;
    mbedtls_ecdsa_init(&ctx);
    mbedtls_ecp_group_load(&ctx.grp, MBEDTLS_ECP_DP_SECP256R1);
    mbedtls_mpi_read_binary(&ctx.Q.X, TRUSTED_PUBKEY_X, 32);
    mbedtls_mpi_read_binary(&ctx.Q.Y, TRUSTED_PUBKEY_Y, 32);
    mbedtls_mpi_lset(&ctx.Q.Z, 1);

    /* 3. Verifier la signature */
    mbedtls_mpi r, s;
    mbedtls_mpi_init(&r); mbedtls_mpi_init(&s);
    mbedtls_mpi_read_binary(&r, meta->sig_r, 32);
    mbedtls_mpi_read_binary(&s, meta->sig_s, 32);

    if (mbedtls_ecdsa_verify(&ctx.grp, hash, 32, &ctx.Q, &r, &s) != 0)
        halt_secure();

    /* 4. Nettoyer et sauter */
    mbedtls_ecdsa_free(&ctx);
    mbedtls_mpi_free(&r); mbedtls_mpi_free(&s);

    /* Desactiver interruptions, remettre SP, sauter a l'application */
    __disable_irq();
    uint32_t app_sp  = *(volatile uint32_t*)APP_START;
    uint32_t app_pc  = *(volatile uint32_t*)(APP_START + 4);
    __set_MSP(app_sp);
    ((void(*)(void))app_pc)();
}

typedef struct {

    uint16_t duration;
    uint8_t  addr1[6];   // destination

static bool is_eapol(const uint8_t *payload, uint16_t len) {
    /* EAPOL : LLC header 0xAA 0xAA 0x03 + OUI 0x00 0x00 0x00 + type 0x88 0x8E */
    if (len < 8) return false;
    return (payload[6] == 0x88 && payload[7] == 0x8E);
}

static void sniffer_cb(void *buf, wifi_promiscuous_pkt_type_t type) {
    if (type != WIFI_PKT_DATA && type != WIFI_PKT_MGMT) return;

    wifi_promiscuous_pkt_t *pkt = (wifi_promiscuous_pkt_t*)buf;
    uint16_t len = pkt->rx_ctrl.sig_len;
    uint8_t  *data = pkt->payload;

    if (is_eapol(data, len)) {
        ESP_LOGI(TAG, "EAPOL frame captured, len=%d", len);
        /* Envoyer la trame brute via UART pour analyse sur PC */
        uart_write_bytes(UART_NUM_0, (char*)&len, 2);
        uart_write_bytes(UART_NUM_0, (char*)data, len);
    }
}

def extract_pmkid(pkt):
    """Cherche un PMKID dans une trame EAPOL Key (4-way handshake, msg 1/4)."""
    if not pkt.haslayer(EAPOL): return
    raw = bytes(pkt[EAPOL])
    # EAPOL Key : descriptor type=2, key info, key length, replay counter, nonce...
    # PMKID dans le RSN IE du msg 1/4 : offset variable, chercher OUI 00:0F:AC:04
    if len(raw) < 99: return

POWER_GPIO = 17   # controle alimentation cible via transistor
TRIG_GPIO  = 27   # lecture trigger cible (optional)

pi.set_mode(POWER_GPIO, pigpio.OUTPUT)
pi.write(POWER_GPIO, 1)   # alimentation ON

uart = serial.Serial('/dev/ttyUSB0', 115200, timeout=1)
time.sleep(1)

def power_glitch(delay_us: int, width_us: int) -> str:
    """Coupe l'alimentation pendant width_us µs apres delay_us µs."""
    time.sleep(delay_us * 1e-6)
    pi.write(POWER_GPIO, 0)          # coupure
    time.sleep(width_us * 1e-6)
    pi.write(POWER_GPIO, 1)          # retablissement
    time.sleep(0.05)
    resp = uart.read(128).decode(errors='ignore')
    uart.reset_input_buffer()
    return resp

# Scanner l'espace de faute
successes = []
for delay in range(0, 500, 25):
    for width in range(1, 30, 2):
        resp = power_glitch(delay, width)
        if "GRANTED" in resp or "bypass" in resp.lower():
            successes.append((delay, width, resp))
            print(f"[!] delay={delay}µs width={width}µs → {resp.strip()}")
        elif len(resp) == 0:
            print(f"    delay={delay}µs width={width}µs → CRASH (reboot)")

print(f"\nSucces total : {len(successes)}")
pi.stop()
uart.close()

2. Coeur AES iteratif (1 ronde / cycle d'horloge)

-- aes_core.vhd
library ieee;
use ieee.std_logic_1164.all;
use ieee.numeric_std.all;

entity aes_core is
  port (
    clk      : in  std_logic;
    rst      : in  std_logic;
    start    : in  std_logic;
    plaintext: in  std_logic_vector(127 downto 0);
    key      : in  std_logic_vector(127 downto 0);
    ciphertext: out std_logic_vector(127 downto 0);
    done     : out std_logic
  );
end entity aes_core;

architecture rtl of aes_core is
  -- Etat interne : 4x4 octets = 128 bits
  type state_t  is array(0 to 3, 0 to 3) of std_logic_vector(7 downto 0);
  type rcon_t   is array(0 to 9)          of std_logic_vector(7 downto 0);

  constant RCON : rcon_t := (
    x"01",x"02",x"04",x"08",x"10",x"20",x"40",x"80",x"1b",x"36"
  );

  signal state   : state_t;
  signal round   : integer range 0 to 10 := 0;
  signal running : std_logic := '0';

  -- Composants S-Box (instancies 16x pour SubBytes en paralllele)
  type byte_arr is array(0 to 15) of std_logic_vector(7 downto 0);
  signal sb_in  : byte_arr;
  signal sb_out : byte_arr;

begin
  -- 16 instances S-Box (combinatoire, substitution parallele)
  GEN_SBOX : for i in 0 to 15 generate
    U_SB : entity work.aes_sbox port map(addr => sb_in(i), dout => sb_out(i));
  end generate;

  process(clk, rst)
    variable s : state_t;
    variable t : state_t;   -- etat temporaire apres ShiftRows
  begin
    if rst = '1' then
      round <= 0; running <= '0'; done <= '0';

    elsif rising_edge(clk) then
      done <= '0';

      if start = '1' and running = '0' then
        -- Charger etat initial = plaintext XOR key (AddRoundKey initial)
        for r in 0 to 3 loop
          for c in 0 to 3 loop
            state(r,c) <= plaintext(127 - (r*4+c)*8 downto 120 - (r*4+c)*8)
                          xor key(127 - (r*4+c)*8 downto 120 - (r*4+c)*8);
          end loop;
        end loop;
        round   <= 0;
        running <= '1';

      elsif running = '1' then
        -- SubBytes : connecte via les 16 instances S-Box
        for r in 0 to 3 loop
          for c in 0 to 3 loop
            sb_in(r*4+c) <= state(r,c);
          end loop;
        end loop;

        -- ShiftRows : decalage ligne i de i positions a gauche
        t(0,0):=sb_out(0);  t(0,1):=sb_out(1);  t(0,2):=sb_out(2);  t(0,3):=sb_out(3);
        t(1,0):=sb_out(5);  t(1,1):=sb_out(6);  t(1,2):=sb_out(7);  t(1,3):=sb_out(4);
        t(2,0):=sb_out(10); t(2,1):=sb_out(11); t(2,2):=sb_out(8);  t(2,3):=sb_out(9);
        t(3,0):=sb_out(15); t(3,1):=sb_out(12); t(3,2):=sb_out(13); t(3,3):=sb_out(14);

        -- MixColumns omis a la ronde finale (round = 9)
        -- AddRoundKey simplifie ici avec la cle brute (expansion separee en vrai AES)
        if round < 9 then
          for r in 0 to 3 loop
            for c in 0 to 3 loop
              state(r,c) <= t(r,c) xor key(127-(r*4+c)*8 downto 120-(r*4+c)*8);
            end loop;
          end loop;
          round <= round + 1;
        else
          -- Ronde finale : assembler le chiffre
          for r in 0 to 3 loop
            for c in 0 to 3 loop
              ciphertext(127-(r*4+c)*8 downto 120-(r*4+c)*8)
                <= t(r,c) xor key(127-(r*4+c)*8 downto 120-(r*4+c)*8);
            end loop;
          end loop;
          done    <= '1';
          running <= '0';
          round   <= 0;
        end if;
      end if;
    end if;
  end process;
end architecture rtl;
Note : cet AES simplifie omet l'expansion de cle et MixColumns pour rester lisible. Pour un AES complet et verifiable avec les vecteurs NIST, voir le projet VHDL open-source vhdl-aes sur GitHub.

F2 · Hardware Trojan FPGA : insertion et detection Avance

Un Hardware Trojan (HT) est un circuit malveillant insere dans un design FPGA ou ASIC. Il reste dormant jusqu'a ce qu'un declencheur rare soit active, puis execute une charge utile (corruption de bit, fuite d'information, blocage). Ce projet insere un trojan dans l'AES du projet F1 et evalue trois strategies de detection.

Prerequis : avoir termine F1 (aes_sbox.vhd + aes_core.vhd). Ce projet ajoute un module trojan au design existant.

1. Architecture d'un Hardware Trojan

ComposantRoleExemple dans ce projet

Charge utile (payload)Action malveillante apres activationInverser le bit 0 du chiffre de sortie

2. Module Trojan en VHDL

-- hw_trojan.vhd
-- Trojan de type "trigger conditionnel + payload discret"
library ieee;
use ieee.std_logic_1164.all;
use ieee.numeric_std.all;

entity hw_trojan is
  port (
    clk        : in  std_logic;
    plaintext  : in  std_logic_vector(127 downto 0);
    ct_in      : in  std_logic_vector(127 downto 0);   -- sortie AES propre
    ct_out     : out std_logic_vector(127 downto 0);   -- sortie apres trojan
    trojan_act : out std_logic                          -- signal d'audit (retire en prod)
  );
end entity hw_trojan;

architecture rtl of hw_trojan is
  -- Trigger : les 2 octets de poids faible du plaintext == 0xDEAD
  -- Probabilite de declenchement : 1/65536 ≈ 0.0015% par chiffrement
  signal trigger : std_logic;
begin
  trigger    <= '1' when plaintext(15 downto 0) = x"DEAD" else '0';
  trojan_act <= trigger;

  -- Payload : si trigger, inverser le bit 0 du chiffre (modification minimale)
  ct_out <= ct_in xor x"00000000000000000000000000000001" when trigger = '1'
            else ct_in;
end architecture rtl;

hw = np.array([bin(v).count('1') for v in range(256)], dtype=np.uint8)

def simulate_fpga_traces(N=5000, key_byte=0x2B, target=0,
                         n_samples=50, noise_std=1.5, seed=42):
    """
    Genere N traces de consommation simulees pour un AES FPGA.
    Modele : P[t] ≈ HW(SBox[pt XOR k]) au cycle SubBytes + bruit.
    """
    rng = np.random.default_rng(seed)
    plaintexts = rng.integers(0, 256, size=(N, 16), dtype=np.uint8)

    # Point de fuite : SubBytes ronde 1, octet TARGET
    leak_val = hw[SBOX[(plaintexts[:, target].astype(np.uint16) ^ key_byte) & 0xFF]]

    # Traces : bruit partout + pic de consommation au cycle SubBytes (echantillon 15)
    traces = rng.normal(4.0, noise_std, size=(N, n_samples)).astype(np.float32)
    traces[:, 15] += leak_val.astype(np.float32) * 2.0   # fuite amplifiee

    return traces, plaintexts

@scared.attack_function
def sbox_hw(plaintext, guesses):
    pt    = plaintext[:, TARGET].astype(np.uint16)

4. Variante materielle : ring oscillator sensor sur FPGA

-- ring_oscillator_sensor.vhd
-- Mesure indirecte de la consommation via la frequence d'un ring oscillator :
-- quand la logique AES consomme plus, la tension d'alimentation baisse
-- et le RO ralentit → compteur RO diminue → on lit une "trace de puissance".
library ieee;
use ieee.std_logic_1164.all;
use ieee.numeric_std.all;

entity ro_sensor is
  generic(N_STAGES : integer := 31);   -- nombre impair d'inverseurs
  port (
    clk_ref  : in  std_logic;          -- horloge de reference (lente)
    sample   : in  std_logic;          -- pulse : capturer le compteur
    ro_count : out unsigned(15 downto 0)
  );
end entity ro_sensor;

architecture rtl of ro_sensor is
  signal ro_chain : std_logic_vector(N_STAGES-1 downto 0);
  signal count    : unsigned(15 downto 0) := (others=>'0');
  attribute ALLOW_COMBINATORIAL_LOOPS : boolean;
  attribute ALLOW_COMBINATORIAL_LOOPS of ro_chain : signal is true;
  attribute DONT_TOUCH : boolean;
  attribute DONT_TOUCH of ro_chain : signal is true;
begin
  -- Chaine d'inverseurs : oscillateur libre
  ro_chain(0) <= not ro_chain(N_STAGES-1);
  GEN_INV : for i in 1 to N_STAGES-1 generate
    ro_chain(i) <= not ro_chain(i-1);
  end generate;

  -- Compteur synchrone sur horloge de reference
  process(clk_ref)
  begin
    if rising_edge(clk_ref) then
      count <= count + 1;
    end if;
  end process;

  -- Capture sur front de sample
  process(sample)
  begin
    if rising_edge(sample) then
      ro_count <= count;
    end if;
  end process;
end architecture rtl;
